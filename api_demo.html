How to Use This API in Your Project

Anyone can use this API to build trading apps, charts, or bots. Just change the asset name!

JavaScript Example:

// Fetch any asset - just change the name!
fetch('http://127.0.0.1:8001/api/price/BTCUSD_otc')
  .then(r => r.json())
  .then(data => {
    console.log(`Price: $${data.price}`);
  });

// Get candle data for charts
fetch('http://127.0.0.1:8001/api/recent/EURUSD_otc?limit=100')
  .then(r => r.json())
  .then(data => {
    // data.candles contains OHLC data
    drawChart(data.candles);
  });

Python Example:

import requests

# Get any asset price
asset = 'BTCUSD_otc' # Change this!
r = requests.get(f'http://127.0.0.1:8001/api/price/{asset}')
data = r.json()
print(f"{asset}: ${data['price']}")

# Get candles for analysis
r = requests.get(f'http://127.0.0.1:8001/api/recent/{asset}?limit=100')
candles = r.json()['candles']
